      • Cloud Level Height [Ignored]
      • Cloud Shape [Ignored]
      • Cloud Pattern

Gls Cloud Cube Type Editor Things

  • Cloud Level Height [Ignored]
    • Stratus (Low)
    • Alto (Medium)
    • Cirrus (High)
    • Cumulus (Vertical)
  • Cloud Shape [Ignored]
    • Stratus [spread]
    • Cumulus [Pile]
    • Nimbus [Rain]
  • Cloud Pattern
    • Random Full (and sometimes a litle outside the box)
    • Linear XorY [___] (all the ducks in a row)
    • Flat XxY [_] (Does not actually work right: looks Linear)
    • Rounded Mass [_] (Similar to Random Full)
    • Open Shape... (Used ONLY with Loaded Shapes)
  • [Generation Method] is what type is used: Individual Sprites, Particle System, Open Shapes, CSprite, Poly Fog, PL Mask, Procedural LOD, Real Slow.
    • Individual Sprites (The Main Man)
    • Particle System (Similar to above, I need to do tests to see if either is 'better..faster' than the other)
    • Open Shapes...(Used ONLY with Loaded Shapes) Actually these are 'Individual Sprites' with the positions specificially determined.
    • PFX CSprite (ONLY 1 allowed) (Does not work well with others)
    • PFX Poly Fog (Cribbed from Erics: The Ball)
    • PFX PL Mask (May be only 1 Allowed, I have not tried 2 yet.. they take a LOT of sprites..CPU) (More than 1 Character requires adjusting the Image Size)
    • Image Sprite: The First texture image is used (with the first IS Cube, then the second with the second,...). Use the Shape Resolution to set the Sprite density. This is different in the number should be the actual Density desired: A 20 will be used as 20x20=400 possible sprites... (The sprites are small so it takes a LOT of sprites..CPU). Not all are used (Black areas are left blank), unless the image is all color. Those Not used are culled.. Setting the Cubes/Size number of parts (sprites) is ignored... but it is filled in with the number actually used, after generation. Actually these are 'Individual Sprites' with the positions specificially determined.
    • Niniane Clouds (Not Available) Clouds Type
    • Real [Slow] (Not Available) Clouds Type

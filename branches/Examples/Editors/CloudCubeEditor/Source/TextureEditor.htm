GLScene Texture Editor

  • General Use
    • The general purpose is to Black out the edge of images so they look more like Clouds than square textures. The most usefull tool for this is the Circle (with the Check boxes Unchecked).. at various sizes. Generally, Less work is required with MF Maker images (if they are done with the check box checked). Procedural and CA Maker images require some 'artistic license' to black out portions...
    • The Top Right portion Loads and Saves the image, located on the Left half. The Bottom Right portion has 2 halves. The Left is Painting and the Right is Magic Wand Filling. Right clicking the panel area to the left will pop-up a Painting menu.
    • Clicking the [Painting Cloning] Toggles between the two activities. Cloning is turned on when the button is pressed. Right clicking the panel area to the right will pop-up a Cloning menu.
    • The image can be saved using the Save Image button. Changing the Image size to larger than 128 (256 or 512) may allow cropping the image to select a portion for use. Numbering the images (for example: a1..a16 or b1..b16) will aid selection into the texture combiner and keep the images in groups.

Texture Editor Things

  • Top Portion
    • [Open] button will activate a file open dialog.
      • The file name is displayed here, after loading.
      • The Green arrow button Undoes the last action.. only 1 level of Undo... it will Flip back and forth.
      • ? Help got you here
      • [Exit] button Closes the form.
    • "Save Image" Button
      • Selecting a Button will activate the Save file dialog to enter the file name. Using BMP as the file type for other than 24 bit may lead to errors if opened in auxiliary image editors, as a 32 bit BMP only has On/Off and these change individual pixel transparency values. Cloud Cubes will actually MAKE the image Transparent, No matter which button is used. (Well, I could have the program load in images and use whatever Transparency the image had... but that would lead to errors I think.) Transparent images could be used in programs to skip the 'Make Transparent' part of loading, as well as having the Images store MipMaps.
      • 24 bit is a normal image with No transparency
        32S is a Super Black Transparent image: Any portion that is Black will be transparent.
        32A Luminance becomes transparent
        32Sqr The Square Root Luminance becomes transparent
  • Cloning
    • Panel, Check box
      • Panel displays Location
      • Check box flips TopLeft to Top or Bottom
    • Cloning will Clone the portion TO the Mouse Area. Pressing the [Shift] key while Cloning will allow moving the Cloned Target area
      • To do: Make the Clone target be outlined..Highlighted
    • Right Clicking the Clone panel will pop-up a menu
      • Size: of the Cloned area
      • Box or Circle: the Shape to be cloned
  • Painting
    • The [5] box controls the size of the Paint tool.
    • The left check box toggles using White or Black as the Paint color.
    • [Paint Box] Clicking the box activates a Color selection dialog.
    • The right check box toggles using [White or Black] or the [Paint Box] color as the Paint color.
    • [Dot, Box, Circle, Flood, Edge, Paint] selects thePaint tool
      • Dot only works if the Size is 1, tho if the size is 1 Circle does about the same thing...
      • Box shape IAW size
      • Circle shape IAW size
      • Flood fill the Image
      • Edge fill the Image
      • Paint an area IAW size with random dots.. sort of a 'Spray Paint' effect.
    • [Smooth] will smooth..blur the image using which filter selected from the box on the right. [The Last button area will perform Edge Detection, rather than smooth.. usefull for 'cleaning' the edge of a cloud.. maybe.
  • Magic Wand Filling Wanding is done using the Right Mouse button

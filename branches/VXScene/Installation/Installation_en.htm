Installation of VXScene
in Embarcadero RAD Studio

The Instruction for installing VXScene library into Embarcadero RAD Studio to work with Delphi & C++Builder includes next steps:

1. Download source codes of VXScene from the URL of repository https://svn.code.sf.net/p/glscene/code to a SVN directory on your disk, e.g. D:\VXScene\SVN_VXScene. Use TortoiseSVN client or embedded subversion control system for RAD Studio environment in menu File | Open From Version Control… to check out the code.

You may also to get a current Snapshot of the trunk or branch from code page http://sourceforge.net/p/VXScene/code/HEAD/tree/  or download official releases from the page http://sourceforge.net/projects/glscene/files/

 

2. Make a copy of the code in a separate directory, e.g. in the working directory D:\code, to prevent original sources from occasional changes. You may skip this step if you don’t need to update the code from SVN repository further.

 

3. Copy auxiliary dynamic libraries from ..VXScene\external into C:\Windows\System32 directory to support 3D sounds, shaders and physics (BASS, FMOD, OpenAL, SDL, CG, ODE, Newton). Otherwise you will need to have ones in every application directory with exe file that calls an appropriate dll. To do this, please, run setup.bat in ..\VXScene\external directory as administrator. Note: on 64-bit OS Windows you may need to copy *.dll  files to C:\Windows\SysWOW64 directory manually.

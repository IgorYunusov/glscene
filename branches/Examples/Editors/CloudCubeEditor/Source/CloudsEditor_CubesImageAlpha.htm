GLScene Gls Cloud Cube Image Alpha Editor

  • General Use
    • Set the Parameters for the Textures: all 24 available images. They are Global: these are set for all Cloud Cubes.
      • There are 4 boxes full of parameters
      • There is a Get and Set box. Clicking the Get number will display the Parameters for that Image (loaded or not). Clicking the Set number will Set the Parameters for that image.
    • I am still working on this and what is what. The Default (top) setting for each is what I use. The second is also good. Some combinations will NOT WORK.

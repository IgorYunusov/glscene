GLScene Gls Cloud Cube Image PL Mask Editor

  • General Use
    • PL Mask Cloud Cube parameters are set on this page. They do not use texture images, Albedo, nor Size. Use of PL Mask is set on the Cubes/Type page.
    • There are 3 types of Masks that can be made: An Alpha character, a loaded image (untested) and a shape created using the Menu available by Right clicking the form.
      • The button at the Top left will load an image into the Top Picture. Its size should match the value of the Image Slider. 20..100 pixels Square.(untested)
      • There are 3 images. The bottom 2 will only be filled in AFTER the PL Mask is actually made.

    Gls Cloud Cube Image PLMask Editor Things

    • The AIM Buttons determine which type of PL Mask is created..Used.
    • Parameters
      • Image Size Slider. 20..100 pixels Square.
      • AIM Buttons: Alpha Character, Image (loaded), Menu shape
      • The 3 sliders set (menu) shape parameters
        • Offset from Max size of the image (so an outline can be made)
        • Line Width
        • shape width
      • Character
      • Size (Font size, should be smaller than image size)
      • These parameters affect any type PL Mask
        • Depth of the PL Mask
        • Scale of the PL Mask
        • Pitch
        • Roll
        • Turn
        • Interval: Rapidity..Density of the Particles of the PL Mask
    • Menu shapes (Black is Void, White is filled with Particles)
      • Sphere (solid
      • Torus (hollow)
      • Cube (solid)
      • Tube (hollow)
      • TV (hollow)
      • Star (hollow)

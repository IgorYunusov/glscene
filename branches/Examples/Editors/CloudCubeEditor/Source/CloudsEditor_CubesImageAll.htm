GLScene Gls Cloud Cube Image All Editor

  • General Use
    • There are 8 pages, This page (has a button to Set the data into the Material Library, A Checkbox to set the use of a 16 portion image, and A Checkbox to set the use of a 64 portion image); 1 to set the type of image the file contains; 1 to set the Alpha properties of the image; 1 to set PL Mask data; 4 to load Texture names toggle their use and set the amount of use.
      • No matter the Cloud type , AT LEAST 1 file must be selected and the Button on the All page pressed.
      • The Type is set for 24 images and is actually global, not changed per cube
      • The Alpha is set for 24 images and is actually global, not changed per cube
      • PL Mask is only used for that 1 Cloud type.
      • The 4 image pages contain a total of 24 possible images to be used for All clouds. The Name is Global, the Percent used and if used is set Per Cube.
    • Cubes: Image Clouds Editor Image Type
    • Cubes: Image Clouds Editor Image Alpha
    • Cubes: Image Clouds Editor Image PL Mask
    • Cubes: Image Clouds Editor Image Texture pages
  • Even though all the states are Stored for 24 Textures, the 16 Sets of 1 Texture and the 64 Sets of 1 Texture, the Sets are not used At All. (still figuring out how to divide the textures and assign them... )

For all this demos are supposed that the directory BCBDemos are in the GLScene directory
and GLScene are in D:\Library\GLScene_BCB6\
1- the header files are placed in D:\Library\GLScene_BCB6\include
2- the not GLScene's lib/dll files are placed in D:\Library\GLScene_BCB6\lib 
3- the dcu/obj files are placed in D:\Library\GLScene_BCB6\dcu

Please, for each project, edit the options and correct the include and lib path 
according to your configuration.

Library directory contains some usefull library

Gls Cloud Cube Image Editor Things

  • Each page has 6 sets of : a Button to activate a File Dialog, its File name area, a Check box, a %Min and %Max data input area
    • The Check box toggles the use of that image.
    • % Min is the least it will be used (> 0), if checked for use.
    • % Max is the most it will be used, if checked for use.

GLScene ProceduralClouds

  • General Use
    • Sharpness and MinCut adjust the Albedo range and Density of the texture.
      • Sharpness: 1..99, not much usable range before it is saturated.
      • MinCut: 0..255
    • I set the Slider to the far left (1 image) and 100%: This displays 1 image at full size.. other settings make 'Tiled' images
    • [12345] is the Cloud Seed Number. Press the [..] button to display the new image. There is no known Seed = Type of image. M87S97
    • The Animated checkbox will process the cloud. Click checked to start and Click again to uncheck..Stop the animation, when it displays a desirable image. The image can be saved used the Save Image button. Changing the Image size to larger than 128 (256 or 512) may allow cropping the image to select a portion for use. Numbering the images (for example: a1..a16 or b1..b16) will aid selection into the texture combiner and keep the images in groups.

Procedural Clouds Things Changing a Value selection [..] will create a new texture.

    • Animated [..] will create new images... Do NOT use Seamless.. it takes a LOT longer and you miss some of the image changes.
    • [..] Inputs all parameters for a new image (some parameters do NOT automatically create a new image.. some do when changed.
    • ? Help got you here
    • Format is the type image.. this requires an RGBA 32 bit.. the A is Alpha : the Transparency, 32 bit makes better images.
    • Compression is the type compression set for the image
    • These settings (Format and Compression) are Not part of the image.. they are information when using the images in a program, to determine the settings desired.
